# PizzaApp (front)

Web application for ordering pizza, which contains a shopping cart. Besides pizza, there are another available items in the menu.
App provide an opportunity for checking out an order, taking delivery address and contact details (phone number).
Login is not required but could be available for checking the history of orders. 
This repository is front-end part of SPA application. Backend and deployment version locates [here](https://github.com/Mi-lex/pizza-back).

## Built With

* [React](https://github.com/facebook/react) - The library used
* [React-router](https://github.com/ReactTraining/react-router)
* [Redux](https://github.com/reduxjs/redux) - State container
* [Redux-saga](https://github.com/redux-saga/redux-saga) - Side effects handler
